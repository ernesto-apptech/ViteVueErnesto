# Ernesto app using VUE + VITE

This a simple web app that has a counter button. This app uses Vue 3 `<script setup>` SFCs and VITE
The app is running in PORT 8080 and is using NodeJS v20

## INSTALLING
- Install all dependencies:
  `npm install`
- Build
  `npm run build`

- Developing mode
  `npm run dev`
